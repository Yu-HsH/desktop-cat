Desktop Cat 배포 폴더

1) 실행 방법
- app\Desktop Cat.exe 실행

2) 포함 내용
- Electron 런타임 + 앱 리소스 전체 포함
- 추가 설치 없이 실행 가능(Windows)

3) 권장 배포 방식
- 이 Desktop-Cat-Deploy 폴더 전체를 압축(zip)해서 전달
- 사용자 PC에서 압축 해제 후 Desktop Cat.exe 실행

4) 주의
- Windows SmartScreen 경고가 나오면 [추가 정보] -> [실행] 선택
- 설정 파일은 사용자 PC의 AppData(userData)에 자동 저장됨
